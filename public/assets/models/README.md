Folder ini disiapkan untuk asset GLB/GLTF lokal. Game tetap berjalan tanpa file-file ini karena memiliki fallback asset prosedural.

Nama file yang akan otomatis dicoba oleh game:
- soldier.glb
- poacher.glb
- elephant.glb
- zebra.glb
- gazelle.glb
- helicopter.glb

Letakkan file GLB di folder ini, lalu jalankan ulang game.
